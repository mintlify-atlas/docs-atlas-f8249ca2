Juego Adivinanza ("Guessing Game" in Spanish) is a simple, fun Java console game where you pick a difficulty level and try to guess a randomly chosen number. With three modes ranging from Easy to Hard, it's a great starting point for learning Java basics like loops, conditionals, and user input.

## How it works

<Steps>
  <Step title="Compile the source file">
    Run `javac Juego.java` in the directory containing the source file to produce `Juego.class`.
  </Step>
  <Step title="Launch the game">
    Run `java Juego` to start the game. The menu appears immediately.
  </Step>
  <Step title="Pick a difficulty">
    Enter `1` for Easy (0–3), `2` for Normal (0–10), or `3` for Hard (0–20).
  </Step>
  <Step title="Make your guess">
    Type a number and press Enter. The game tells you instantly if you're right or wrong — and reveals the answer either way.
  </Step>
</Steps>

<Tip>
  After each round the menu reappears automatically so you can play again or switch difficulty. Enter `4` at any time to exit.
</Tip>
