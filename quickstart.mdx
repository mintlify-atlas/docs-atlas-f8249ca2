This page walks you through everything you need to do to compile Juego Adivinanza from source and play your first round. You will go from an empty directory to a running game in under two minutes.

## Prerequisites

- **Java JDK 8 or later** — you need both `java` (the runtime) and `javac` (the compiler). Run `java -version` and `javac -version` to confirm they are available.
- **A terminal** — macOS Terminal, a Linux shell, or Windows Command Prompt / PowerShell.

<Note>
  If `javac` is not found, your `JAVA_HOME` environment variable may not be set, or the JDK `bin` directory may not be on your `PATH`. Install a JDK (version 8 or later) and follow its setup instructions to add `javac` to your `PATH`.
</Note>

## Install and run

<Steps>
  <Step title="Clone the repository">
    Download the source code with Git:

    ```bash
    git clone https://github.com/EduCabrera-k/Juego_Adivinanza.git
    ```

    If you prefer not to use Git, download the ZIP from GitHub and extract it.
  </Step>
  <Step title="Navigate to the project directory">
    ```bash
    cd Juego_Adivinanza
    ```
  </Step>
  <Step title="Compile the source file">
    ```bash
    javac Juego.java
    ```

    This produces `Juego.class` in the same directory. You only need to run this step once — or again after any change to the source file.
  </Step>
  <Step title="Run the game">
    ```bash
    java Juego
    ```

    The main menu appears immediately in your terminal.
  </Step>
</Steps>

## Playing your first round

After running `java Juego` you will see the main menu:

```text
ADIVINA EL NUMERO
Escoje un Modo
1 ------- Modo Facil (0-3)
2 ------- Modo Normal (0-10)
3 ------- Modo Dificil (0-20)
4 ------- Salir
```

Enter `1` to select Easy mode (numbers 0–3). The game then prompts you for a guess:

```text
Cual numero crees que es?
```

Type a number and press Enter. If your guess is correct:

```text
Correcto, el numero es: 2
```

If your guess is wrong:

```text
Fallaste, el numero es: 2
```

Either way the correct number is revealed and the menu reappears so you can play again.

<Tip>
  The game loops automatically after every round. When you are done playing, enter `4` at the main menu to exit.
</Tip>
